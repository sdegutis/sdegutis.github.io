Introducing InspectorKit It's been a week or so since I've released the source to [my Inspector panel](/blog/2009-02-26-inspector-panel-for-leopard/) for Mac OS X (Leopard only). Well, using that code was a little... inconvenient, for a number of reasons. So, I've made it a lot more convenient for developers to use. Introducing, InspectorKit! InspectorKit is both a framework and an Interface Builder plugin. You can check out the source on github. This is basically a major update to the Inspector panel, as it packages all the code into a convenient framework for developers to include in their projects, and allows them to test the inspector at design time.
