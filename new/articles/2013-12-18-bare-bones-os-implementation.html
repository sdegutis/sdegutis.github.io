Bare bones OS implementation [This tutorial on writing a bare-bones OS](http://wiki.osdev.org/Bare%5FBones) looks like a really cool little project. It seems to only use generic VGA functions ("the VGA text mode buffer") as output, which is kind of exciting. The minimalist in me kind of doesn't like that it relies on GRUB and a few other external tools, but then I just remind myself that a bootloader is not the same as an OS, and not nearly as interesting. Plus I'm sure there are more minimalist bootloaders out there that I can swap out if I really wanted to.
