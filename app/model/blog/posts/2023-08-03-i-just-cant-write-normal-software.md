title: "I just can't write normal software."
---

All of the projects I've been successful at, whether open source in my free time, or paid as a contractor to write, have been weird.

* Firmware in C for the ultra minimalist & programmable Atreus keyboard

* A multiplayer pixel (100x100) paint websocket app that can create timelapse gifs via a chat command

* An animating web app in TypeScript utilizing pure DOM that works beautifully with a stylus

* A Lua/Love2d library that utilizes graphical assets created from pico-8 and wraps them in a convenient API

For some reason, I can make a ton of progress really fast on weird apps like these, but I can't make rapid progress on a traditional LAMP stack or Express.js app.

The good news is that I've figured out my niche: as a software engineer, I'm able to solve really weird problems. If you have one, I'm available so [let's talk](/contact.html).
